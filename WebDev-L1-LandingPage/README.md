# PowerUp - Fitness App Landing Page

## Objective
A static landing page built for "PowerUp" — a fictional fitness app — as part of the Oasis Infobyte Web Development Internship (Level 1, Task 1).

## Tech Stack
- HTML5
- CSS3 (Flexbox, Media Queries)

## Features
- Sticky navigation bar with 4 links
- Hero section with headline, subheadline, and CTA button
- Features section with 3 highlighted cards
- About section
- Footer with social links
- Fully responsive design (mobile, tablet, desktop)
